Flood Fill
A plane segmentation algorithm for lidar point clouds designed for floor plan matching (with code)..
  • An efficient framework that can achieve robust and long-term localization, given a floor plan as prior, without being affected by position available objects. This framework provides a good case study on how to fully utilize prior information for localization (with code).
  • FP-Recov: A simple and effective algorithm for extracting semantic information from floor plan images.
  • Flood Fill: A plane segmentation algorithm for lidar point clouds designed for floor plan matching (with code).
  • ANNF: A novel data structure in the form of an approximate nearest neighbour field (ANNF) which enables an efficient look-up of the nearest geometric floor plan elements (e.g. wall segments) for any given point in the localization space.
  • A virtual dataset pipeline designed to test proposed algorithm. This pipeline can automatically convert SVG format floor plans into 3d models in gazebo, and preserve connectivity information such as doors, generate clutter according to rules. At the same time, this pipeline can also automatically plan paths in the model according to hand-drawn points and simulate the motion of lidar and IMU, and obtain simulated sensor data (with code).
The Framework
Given a floor plan, we can perform long-term localization without being affected by position available furniture. First, we use FP-Recov to recover semantic information from the given floor plan image and convert it to Scalable Vector Graphics (SVG) format. Then, we construct an ANNF to quickly find the nearest neighbor geometric element. At the same time, we use Flood Fill to segment the point cloud data and obtain usable planes. We use these planes to predict and maintain gravity, calculate the 3-DOF pose, and perform matching with the ANNF to obtain the other 3-DOF pose, and finally perform joint optimization.
